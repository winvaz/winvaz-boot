What Is Apache Hadoop?
The Apache™ Hadoop® project develops open-source software for reliable(可靠性), scalable(可扩展性), distributed computing(分布式系统).

解决问题：
海量数据的存储（HDFS）
海量数据的分析（MapReduce）
资源管理调度（YARN）
